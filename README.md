## How to Use
1. Enter or paste your text
2. Select a voice
3. Adjust speed/pitch if needed
4. Click "Speak Now" to listen
5. Use pause/resume/stop controls
6. (Optional) Click download for audio file

## Host on GitHub Pages
1. Fork this repository
2. Enable GitHub Pages in settings:
   - Go to Settings > Pages
   - Source: "Deploy from a branch"
   - Branch: "main" (or "master")
   - Folder: "/root"
3. Your site will be live at: `https://yourusername.github.io/repository-name`

## Limitations
- **Download feature is simulated** - For actual WAV downloads, you'd need additional server-side code
- Works best in modern browsers (Chrome, Edge, Firefox)
- Voice availability depends on your browser/OS
- Limited to ~5000 characters for best performance

## Browser Support
- Chrome 33+
- Edge 14+
- Firefox 49+
- Safari 7+
